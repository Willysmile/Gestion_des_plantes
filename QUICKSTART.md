# 🚀 DÉMARRER MAINTENANT

## ⚡ En 2 Minutes Montre en Main

```bash
# 1. Terminal 1 - Backend
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload

# 2. Terminal 2 - Frontend  
cd frontend
npm install
npm run dev

# 3. Ouvrez http://localhost:5173
# C'est prêt ! 🎉
```
